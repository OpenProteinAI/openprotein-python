class openprotein.api.embedding.EmbeddingResultFuture(session: APISession, job: Job, sequences=None, max_workers=10)[source]

This class defines a future result from an inference request. Results are viewed as a mapping from sequences to result arrays, which can be queried using the Map interface. The status of the job can be checked using the AsyncJobFuture interface and all results can be retrieved as a list of tuples of (sequence, result) pairs from the .get() function. Individual results can be retrieved with __getitem__(sequence) and the object can be iterated like a dictionary. Results are cached so repeated indexing of the same result will not result in additional GET requests against the server.
